# Things I Did

Activity log. Claude updates this as you work — timestamps, what happened, what was decided. You never have to manually track what you did.

## How It Works
- Each day gets a `## YYYY-MM-DD` header
- Under that, bullet points with timestamps and brief descriptions
- Claude adds entries automatically during conversations
- Start of next session, Claude reads this to know where you left off

## Getting Started
Just start working with Claude. Say things like "log that we set up the database" or let Claude pick up the habit naturally. After a few sessions, this file becomes your work diary.
