# Learning Tracker

Track concepts through three stages: Queue (want to learn), Active (currently studying), and Solid (confident enough to use).

## Queue
Things you want to learn but haven't started yet.

## Active
Currently studying or practicing. Move here when you start working with a concept.

## Solid
You've used it in practice and could explain it to someone else. Move here when it clicks.

## How It Works
- Add topics as you encounter them: "add MCP to my learning queue"
- Promote topics as you gain confidence
- Claude references this to know your skill level on different topics
- Helps Claude calibrate explanations — more detail for Queue items, less for Solid ones
